# surgical_tool_tracker
ENGG1101 — Engineering Challenges | The University of Hong Kong | January 2026 – May 2026

- Description: 
    An autonomous tracking system that uses computer vision to detect and monitor surgical instruments in real time, designed to assist surgeons by       providing continuous tool visibility during procedures.

- Features: 
    Real-time detection — YOLO-based computer vision model detects and tracks surgical tools via a Raspberry Pi-mounted camera
    Motorized tracking — 2-DOF servo-actuated camera mount automatically follows detected instruments
    Alert system — OLED status display and buzzer notify surgeons when a tool exits the camera's field of view
